Hardware Requirements:
1.Raspberry Pi 3 B/B+
2.Passive Infrared Sensor(PIR)
3.2 Ip Webcam/camera

Software Requirements
1.Raspbian OS
2.OpenCv
3.Ip webcam(on the phone)
4.python 2 or 3


This script allows the usage of two motion sensors and two cameras to create a motion sensing security system.Each camera is conrolled by a single pir sensor.
